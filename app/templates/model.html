Confussion Matrix of Proposed Model

This figure shows that the model is very accurate in detecting the class “Not Safe,” with 1401 correct predictions and only 16 errors. For the class "Safe," the model produced 155 correct predictions but still had 27 errors. Even though the data was balanced using SMOTE oversampling, the model was still superior in detecting "Not Safe." This may be due to the complexity of the features in the "Safe" class, so further analysis is needed to improve the prediction accuracy in this class.

Accuracy Comparison

Based on the graph, LGBM combined with the “RFE SMOTE OPTUNA” method gives the highest accuracy of 98.01%, indicating the combination of LGBM with RFE, SMOTE, and OPTUNA is very effective for Water Quality Index classification. The "SMOTE" method has the lowest accuracy, which is 97.15%. Overall, all methods produced high accuracy (above 97%), with the combination of techniques tending to improve model performance.

Classification Report of Proposed Model

The model performed very well for the “Not Safe” class as the precision and recall were close to 1. However, the performance for the “Safe” class was slightly lower, indicating potential data imbalance or difficulty for the model to detect the class. The overall accuracy reached 97.31%.

Average Cross-validation of Proposed Model

The model performed very well and consistently on the training data based on the average cross-validation score. This indicates that the model has good generalization to the training data and can be expected to perform well on unseen data, although it needs to be further tested to avoid potential overfitting.
